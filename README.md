# Smart Class App

## Overview

Smart Class App is a Flutter MVP application that allows students to check into class using QR code scanning and GPS verification.

## Features

* QR Code scanning for attendance
* GPS location capture
* Check-in reflection form
* Finish class review screen

## Screens

1. Home Screen
2. Check-in Screen
3. Finish Class Screen

## Installation

1. Clone repository

```
git clone https://github.com/yourusername/smart_class_app.git
```

2. Install dependencies

```
flutter pub get
```

3. Run the app

```
flutter run
```

## Dependencies

* Flutter
* Geolocator
* Mobile Scanner

## Firebase Deployment

The project can be deployed using Firebase Hosting.

```
firebase deploy
```
